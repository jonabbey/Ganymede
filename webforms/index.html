Notes

Most users on Windows should use the 'Java Plugin' option to run the Ganymede clients. However, if you have installed the Java Web Start software on your machine, you should use the 'Java Web Start' option, which will let you keep a copy of the Ganymede clients on your local system for fast start up time.

If you are on a Macintosh or OS/2 system, or if you are running Mozilla on any platform, your browser may support the 'Native' option, which lets the browser load its own JDK 1.2 compatible JVM to handle the Ganymede clients.
